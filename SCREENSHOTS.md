## Color Schemes

### Status Badge Colors

**Blazor Web App:**
- Draft/Pending: Gray (`secondary`)
- Sent/Scheduled: Blue (`info`)
- InProgress: Blue (`primary`)
- Accepted/Completed: Green (`success`)
- Rejected/Cancelled: Red (`danger`)
- Expired/OnHold: Orange (`warning`)

**React Native App:**
- Same color scheme for consistency
- Uses iOS/Android native styling

### Priority Colors
- Low: Gray
- Normal: Blue
- High: Orange
- Urgent: Red

## Running the Applications

### Blazor Web App
```bash
cd src/AccountsPOC.WebAPI && dotnet run
cd src/AccountsPOC.BlazorApp && dotnet run
# Navigate to http://localhost:5193
```

### React Native App
```bash
cd ContractorApp
npm install
npm start
# Press 'w' for web, 'i' for iOS, 'a' for Android
```
