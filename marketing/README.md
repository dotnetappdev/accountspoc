## Creating Actual Screenshots

To capture real screenshots from the app:

### iOS Screenshots
1. Run app on iOS simulator or device
2. Press `Cmd + S` to capture screenshot
3. Recommended resolutions:
   - iPhone 6.7": 1290 x 2796
   - iPhone 6.5": 1242 x 2688
   - iPhone 5.5": 1242 x 2208
   - iPad Pro 12.9": 2048 x 2732

### Android Screenshots
1. Run app on Android emulator or device
2. Press `Ctrl + S` or use device screenshot
3. Recommended resolutions:
   - Phone: 1080 x 1920 or 1440 x 2560
   - Tablet: 1536 x 2048 or 2048 x 2732

### Web Screenshots
1. Open app in browser
2. Use browser developer tools
3. Set device toolbar to mobile size
4. Recommended: Chrome DevTools screenshot tool

### Tips for Great Screenshots
- Use realistic data (not Lorem Ipsum)
- Show key features in action
- Include both light and dark modes
- Demonstrate offline indicators
- Show biometric prompts
- Capture before/after states
- Include loading and success states
