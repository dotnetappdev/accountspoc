# Screenshots Documentation

This directory contains screenshots for both the Blazor Web Application and MAUI Mobile Application.

## Directory Structure

- **`blazor/`** - Screenshots from the Blazor web application
- **`mobile/`** - Screenshots from the MAUI mobile application
